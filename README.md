# GPX Route Animator v1.0.0

> **Offline-first GPX mapper for Kenya**  
> Dark mode • EAT time • Latest-first • No internet needed

![Preview](https://img.shields.io/badge/status-v1.0.0-green)  
![License](https://img.shields.io/badge/license-MIT-blue)  
![Python](https://img.shields.io/badge/python-3.8%2B-blue)

A lightweight, open-source tool to **visualize GPX routes** with **offline maps**, **speed heatmaps**, and **Kenya local time (EAT)**. Perfect for runners, drivers, cyclists, and field teams.

---

## Features

| Feature | Description |
|--------|-------------|
| **Offline Maps** | Use `.mbtiles` (no internet) |
| **Dark Mode** | GUI + map tiles |
| **EAT Time** | All timestamps in Kenya local time (`+03:00`) |
| **Speed Heatmap** | Color-coded by speed (0–50+ km/h) |
| **Latest First** | Files sorted newest to oldest |
| **Multi-Select** | Generate many maps at once |
| **Stats Footer** | Total time, distance, stop % |
| **Toggleable Legend** | Speed guide on map |
